Natural++ Documentation Docs

The programming language designed to run like pure human English natively inside a custom C++ Engine.

üí° Philosophy: Natural++ removes curly braces {}, semicolons ;, and confusing traditional syntaxes replacing them with logical, spoken sentences.

1. Comments

Sometimes you need to write notes in your code for yourself or other humans. These are ignored by the compiler. Start a line with note:. 

note: This is a comment. It will not execute.
note: Use comments to document your logic.

2. Variables and Constants

Everything needs to be stored somewhere. You can create variables (storage spaces) to hold numbers or text (strings).

Creating a Variable

Use create variable [name] equal to [value]. 

create variable count equal to 10
create variable user_name equal to "Vimlesh"

Updating an Existing Variable

Use set [name] to [value].

set count to 20
set count to count plus 5

3. Displaying Output

Want to write data out to the terminal? You can use the display command. 

display "Hello, World!"
display user_name
display count

4. Logic / If Statements

Programs need to make decisions. Natural++ reads like a conversation when you use if ... then statements. 

create variable age equal to 18

if age is greater than or equal to 18 then
    display "You are an adult."
otherwise
    display "You are a minor."
end if

Conditions Explained

English Syntax Translates to Traditional Math
is equal to ==
is not equal to !=
is greater than >
is less than <
is greater than or equal to >=
is less than or equal to <=

5. Loops (Repeating Actions)

If you need to iterate, Natural++ currently supports two robust loop patterns: While Loops and Repeat Times.

Repeat X Times

If you know exactly how many times you want a block to execute:

repeat 5 times
    display "Knock knock!"
end repeat

While Condition Is True

Continues to repeat until a logic check becomes false:

create variable money equal to 0

while money is less than 100 do
    display "Working..."
    set money to money plus 20
end while

display "Done working!"

6. Math Operations

Mathematical calculations use standard spoken terms.

  • plus (+)
  • minus (-)
  • times (*)
set score to count times 10
set health to health minus 5

7. Putting It All Together

Here is a complete program that utilizes variables, maths, conditionals, and loops organically: 

note: The Classic FizzBuzz (Modified for Natural++)

create variable number equal to 1

while number is less than or equal to 5 do
    if number is equal to 3 then
        display "It is three!"
    otherwise
        display number
    end if
    
    set number to number plus 1
end while
