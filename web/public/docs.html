Natural++ is a revolutionary programming language that runs natively in C++ but looks like standard written English.

Our completely free online tutorial will teach you how to write logic, create applications, and solve problems without memorizing confusing syntax like brackets {} and semicolons ;.

Example in Natural++

Natural++ Introduction

What is Natural++?

  • Natural++ is a high-level compiled language.
  • It was designed to help beginners and non-programmers bridge the gap into computer logic.
  • It runs natively on a custom C++ Virtual Machine, meaning it's incredibly robust and isolated.

Why Use Natural++?

  • Readable: It reads exactly like a human sentence.
  • Less Error-Prone: No missing semicolons or mismatched curly brackets to worry about.
  • Web Integrated: Test and write code instantly in the browser using the Web IDE.

Natural++ Get Started

To get started with Natural++, you don't need to install anything! You can use our Web IDE directly to write and execute code in real time.

If you prefer a local setup, the language comes compiled via Git & Make (C++ backend). But for this tutorial, we focus on the raw syntax.

Natural++ Syntax

A Natural++ script is executed line by line. Every line represents an instruction.

Example

display "Hello World!"

Omitted Characters

Unlike languages such as C++ or JavaScript, Natural++ does not use:

  • Semicolons ; at the ends of lines.
  • Parentheses () around conditions.
  • Curly brackets {} to group code blocks. (We use readable keywords like end if instead).

Natural++ Output (Print)

In Natural++, you output text or values using the display or show keyword.

Example (Outputting Text)

display "I am learning Natural++"
show "It is very easy to read!"

You can also display math or numbers directly without quotes.

display 300

Natural++ Comments

Comments can be used to explain Natural++ code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

Single-line Comments

To create a comment, simply start the line with note:.

Example

note: This is a comment
display "Hello World!" note: You cannot put note at the end of a line in Natural++ yet. It must be at the start!

Natural++ Variables

Variables are containers for storing data values.

In Natural++, variables are created the moment you declare them using create variable.

Declaring Variables

To create a variable, specify the name and assign it a value using equal to: 

create variable variableName equal to value

Example

create variable name equal to "John Doe"
create variable age equal to 35

display name
display age

Assigning / Updating Variables

If a variable already exists, you change its value using set and to. 

create variable x equal to 10
set x to 20  note: x is now 20

Constants

Sometimes you want to declare a value that makes your intention clear. You can use create constant. 

create constant pi equal to 3.14159

Natural++ Data Types

A variable in Natural++ can store different types of data under the hood. Currently, the most used ones are:

  • Numbers: 10, 3.14
  • Strings: "Hello"

Example

create variable myNum equal to 5               note: Number
create variable myFloat equal to 5.99          note: Floating point number
create variable myText equal to "Hello"        note: String

Natural++ Booleans

Very often, in programming, you will need a data type that can only have one of two values, like:

  • YES / NO
  • ON / OFF
  • TRUE / FALSE

In Natural++, Boolean expressions return true or false implicitly when you use the comparison operators.

Example

create variable x equal to 10
create variable y equal to 9

note: This will evaluate internally to true
if x is greater than y then
    display "Math works!"
end if

Natural++ Conditions and If Statements

Natural++ supports the usual logical conditions from mathematics natively into English statements.

You can use these conditions to perform different actions for different decisions.

  • Use if ... then to specify a block of code to be executed, if a specified condition is true
  • Use otherwise to specify a block of code to be executed, if the same condition is false

The if ... then Statement

if condition then
    note: block of code to be executed if the condition is true
end if

Example

create variable my_age equal to 20

if my_age is greater than or equal to 18 then
    display "You can vote!"
end if

The otherwise Statement (Else)

if condition then
    note: block of code to be executed if the condition is true
otherwise
    note: block of code to be executed if the condition is false
end if

Example

create variable time equal to 20

if time is less than 18 then
    display "Good day."
otherwise
    display "Good evening."
end if

Natural++ While Loop

Loops can execute a block of code as long as a specified condition is reached.

Loops are handy because they save time, reduce errors, and they make code more readable.

The while ... do Loop

The while loop loops through a block of code as long as a specified condition is true: 

while condition do
    note: code block to be executed
end while

Example

In the example below, the code in the loop will run, over and over again, as long as a variable (i) is less than 5: 

create variable i equal to 0

while i is less than 5 do
    display i
    set i to i plus 1
end while
Note: Do not forget to increase the variable used in the condition (set i to i plus 1), otherwise the loop will never end!

Natural++ Repeat Loop

When you know exactly how many times you want to loop through a block of code, use the repeat ... times loop instead of a while loop. 

repeat amount times
    note: code block to be executed
end repeat

Example

This will print "Hello" exactly 3 times:

repeat 3 times
    display "Hello"
end repeat
